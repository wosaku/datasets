# Data sets


  * crimes_original_data.zip: The data comes from the <a href="http://data.vancouver.ca/datacatalogue/crime-data.htm
" target="_blank">Vancouver Open Data Catalogue</a>. 

It was extracted on 2017-07-18 and it contains 530,652 records from 2003-01-01 to 2017-07-13.


  * crimes_clean.zip: The clean and transformed version of the crime data set.
